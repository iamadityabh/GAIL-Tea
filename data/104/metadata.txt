The trainee, Ananya Rao, is currently undergoing industrial training at the GAIL Training Institute, Noida, and is working on a project titled “AI-Powered Employee Support Assistant” under the supervision of mentor Ms. Neha Sharma, Assistant Manager (IT). The project focuses on developing an intelligent chatbot capable of answering common employee queries and improving accessibility to organizational information.

The project consists of five phases: Requirement Analysis, System and Conversation Design, Chatbot Development and Integration, Testing and Response Optimization, and Deployment with Performance Monitoring. The trainee is currently working in the third phase, Chatbot Development and Integration.

During the completed phases, common employee information requirements were studied, conversation flows were designed, database requirements were identified, and the overall chatbot architecture was prepared. Ananya is currently responsible for developing Python backend modules, implementing chatbot APIs, integrating SQL databases, managing user queries, testing response workflows, and maintaining project documentation.

The technologies used include Python, Flask, SQL, Java, Git, REST APIs, and Large Language Model services. The project’s strengths include faster information retrieval, reduced repetitive administrative queries, centralized access to organizational information, and a simple conversational interface. Current limitations include dependency on available organizational data, possible incorrect chatbot responses, and the requirement for continuous testing.

Future plans include implementing Retrieval-Augmented Generation, improving response accuracy, introducing role-based information access, and deploying the chatbot on the organization’s internal network. The project is helping the trainee gain experience in AI applications, backend development, database integration, API development, chatbot systems, and software project management.
