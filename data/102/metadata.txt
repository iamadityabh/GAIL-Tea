The trainee is currently undergoing industrial training at DRDO, Noida, and is actively involved in the development of a web-based Project and Resource Management Portal designed to streamline internal administrative operations and improve project monitoring across various departments. The project aims to provide a centralized platform for managing employee records, project documentation, task assignments, resource allocation, and progress tracking. The trainee is presently working in the second phase of the project, namely System Design and Architecture Planning, under the guidance of mentor Mr. Ramesh Kulkarni, Senior Technical Officer.

The project is being executed through five structured phases: Requirement Analysis, System Design, Development and Integration, Testing and Quality Assurance, and Deployment with Maintenance Support. During the initial phase, extensive discussions were conducted with departmental stakeholders to identify operational challenges and define system requirements. The current design phase involves preparing wireframes, workflow diagrams, database schemas, and security specifications to ensure scalability and maintainability of the application.

The web portal is planned to include modules such as user authentication, project status dashboards, resource utilization monitoring, document management, automated email notifications, and report generation. The trainee has contributed to preparing UML diagrams, designing database tables, documenting functional specifications, and evaluating technology stacks suitable for implementation. Weekly progress review meetings are conducted every Friday to assess milestones and discuss technical challenges.

The project team consists of four members working collaboratively using version control systems and project management tools. The trainee has also assisted in preparing design documents, maintaining requirement traceability records, and conducting preliminary usability assessments. Future phases will involve backend development, API integration, security testing, and deployment on an internal DRDO server environment. Through this training program, the trainee is gaining practical exposure to software engineering practices, web application architecture, database design principles, project documentation standards, and collaborative development methodologies in a professional research and development setting.
