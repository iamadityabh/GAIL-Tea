The trainee, Aditya Khanna, is currently undergoing industrial training at the GAIL Training Institute, Noida, and is working on a project titled “IoT-Based Industrial Equipment Monitoring System” under the supervision of mentor Mr. Rajeev Bhatia, Senior Manager (Instrumentation). The project aims to develop an intelligent monitoring system capable of collecting equipment sensor data, identifying abnormal operating conditions, and providing timely alerts to maintenance personnel.

The project follows a five-phase development lifecycle consisting of Requirement Analysis, Sensor and System Design, IoT Integration and Development, Testing and Performance Evaluation, and Deployment and Monitoring. The trainee is currently working in the third phase, IoT Integration and Development. The initial phases involved identifying equipment monitoring requirements, selecting suitable sensor parameters, designing data collection workflows, and planning communication between IoT devices and the backend monitoring system.

Aditya’s current responsibilities include developing Python-based data processing modules, integrating IoT sensor data, implementing basic anomaly detection logic, testing device communication, maintaining technical documentation, and analyzing system performance. The project uses Python, Embedded C, Linux, IoT communication protocols, and data visualization technologies.

The project’s major strengths include real-time equipment monitoring, early identification of abnormal operating conditions, reduced manual inspection requirements, and centralized access to equipment data. Current limitations include dependency on reliable sensor connectivity, possible false alerts, hardware maintenance requirements, and limited testing with large-scale industrial data.

Future work includes improving anomaly detection accuracy, integrating predictive maintenance capabilities, developing advanced dashboards, and deploying the system within a controlled industrial environment. Through the project, the trainee is gaining practical experience in IoT systems, embedded programming, industrial monitoring, Python development, system integration, and technical project management.
